## Dataset Information

The dataset contains:
Order Date & Region
Customer Segment
Product Category & Sub-Category
Sales & Profit
Order Quantity
Shipping Mode

Data cleaning, preparation, and visualization were all done in Excel using pivot tables, charts, slicers, and calculated KPIs.
